WEB110 Final Project

Project Summary

In this activity, I learned the fundamental principles of CSS, including selectors, declarations, and the box model. By understanding how CSS separates style formatting from HTML and controls the layout and appearance of web pages, I gained insight into how to style HTML elements effectively. Through hands-on exercises and examples, I developed a deeper understanding of CSS syntax and its application in creating visually appealing and responsive designs. In this activity, I will be able to apply my newfound knowledge of CSS to enhance the design and layout of web pages in my current work environment. By leveraging CSS to control styling, layout, and responsiveness across different devices, I can ensure a consistent and engaging user experience for our website visitors, ultimately contributing to the success of our projects.
