Session 12 - Internet Fundamentals/JavaScript

Activity 1: JavaScript - Extra Credit Assignment

In this activity, I learned to manipulate HTML elements using JavaScript. By adding an empty top-level heading and dynamically displaying my name, along with incorporating the current date and time, I honed my scripting skills. Furthermore, drawing shapes on canvas enhanced my understanding of graphical rendering. This knowledge can be applied to enrich user interfaces in various web development projects. In this activity, I will be able to apply dynamic webpage updates, real-time data integration, and canvas manipulation for enhanced interactivity. These skills can elevate internal tools or client-facing applications in my current work environment, fostering better user experiences and more engaging digital solutions.
